# AI Session Archive

This directory contains session logs from AI coding agents working on this project.

## File Naming Convention

- `YYYY-MM-DD-feature-[feature-name].md` – new feature implementation
- `YYYY-MM-DD-fix-[issue-description].md` – bug fixes
- `YYYY-MM-DD-refactor-[component-name].md` – code refactoring
- `YYYY-MM-DD-setup-[tool-or-config].md` – tooling/configuration changes

## Each log captures

- **What changed** – files and scope
- **Why** – decision rationale and tradeoffs
- **Context** – anything a future agent needs to continue the work

---

*This archive helps maintain continuity across AI-assisted development sessions.*
